     RC5-32/12/8 is a type of encryption. Encryption is a way to scramble data so that only one person (or computer) can descramble it. RSA Data Security has sponsored a challenege to find the correct key out of 264. The object of the RSA Challenege is to show that current encryption techniques just aren't good enough.
     WE ARE PART of the Bovine effort to crack the code. The Bovine Keyserver sends each member of the team a range of blocks (230 keys) to crack. We then use their program to calculate each key. If one of them is the right one, RSA will give out $10,000. Six thousand dollars for a charity, $2,000 for Bovine, and $2,000 for us.
     THERE WERE OTHER groups trying to crack the code as well. None of them have decided to take up RC5-64 though.
